# 42576. 완주하지 못한 선수 

## 풀이1
🟩동명이인이 존재할 수 있기 때문에 테이블 객체에 인원수를 기록  
```JS
const table = {
  참가자1: 인원수,
  참가자2: 인원수,
  ...
}
```
- `completion`을 순회하면서 `table`인원수를 1씩 제거
- `table`을 순회하면서 인원수가 1인 사람을 반환

## 풀이2
- `participant`와 `completion`을 정렬한 뒤, 인덱스별로 비교해서 다른 값이 있으면 `participant[index]`를 반환
- 풀이1보다 효율 안좋게 나옴 (코드는 매우 짧음)
